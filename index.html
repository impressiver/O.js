O.js

Ian White

O.js is a Javascript web application framework and object manipulation library that makes it easy to combine prototypical inheritance (extend) and object composition (mixin).

O.js is still experimental. All the things may (and probably will) change.

Getting Started

  1. Download O.js (TAR or via Github) and put the files somewhere within your project resource path.
  2. Include the JS for jQuery and O.js at the end of the BODY: 
    <body>
  ...
  <script src="/path/to/js/jquery.js"></script>
  <!-- Right about here should do the trick -->
  <script src="/path/to/js/O.js"></script>
</body>
    Heads up! O.js also works with your favorite browser-based AMD script loaders, like RequireJS.
